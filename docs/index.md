# BookShopApp Github Pages

## Overview

BookShopApp is a mobile iOS e-commerce application developed using Swift and SwiftUI.

The application allows users to browse books and stationery products, manage shopping carts, place orders, select payment methods, and choose pickup points using map integration.

The project is developed as a team software engineering project for the course “Mobile Programming Technologies”.

The application follows:
- Apple Human Interface Guidelines
- MVVM architecture
- Git workflow
- CI/CD practices
- UML-based software design
