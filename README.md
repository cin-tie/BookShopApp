# BookShopApp

## Description
BookShopApp is an iOS application for ordering books and stationery products.
Users can browse products, manage cart, place orders, choose payment methods,
and select nearest pickup points using map integration.

## Features
- User authentication
- Product catalog
- Shopping cart
- Order management
- Online payment
- Pickup points map
- Localization
- SQLite storage

## Technologies
- Swift
- SwiftUI
- MVVM
- SQLite
- UserDefaults
- GitHub Actions

## Installation

### Steps
1. Clone repository
2. Open `.xcodeproj`
3. Run project
